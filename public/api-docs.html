RDM КГТУ — REST API

OpenAPI 3.0 v1.0.0 Base URL: http://localhost:3000
🔐 Аутентификация
POST /api/login Вход пользователя, возвращает JWT-токен
ПолеТипОписание
emailstringEmail пользователя required
passwordstringПароль required
{ "token": "abc123...", "user": { "id": 2, "name": "Researcher A", "role": "researcher" } }
POST /api/register Регистрация нового пользователя
ПолеТипОписание
namestringПолное имя required
emailstringEmail required
passwordstringПароль, мин. 6 символов required
POST /api/logout Завершение сессии 🔒 Bearer
📦 Наборы данных
GET /api/datasets Список опубликованных наборов данных FTS5 поиск
ПараметрТипОписание
qstringПолнотекстовый поиск (FTS5)
accessopen|restrictedФильтр по доступу
licensestringФильтр по лицензии
pageintegerНомер страницы (default: 1)
limitintegerЗаписей на страницу (default: 12)
{ "total": 4, "page": 1, "limit": 12, "pages": 1, "items": [ {...} ] }
GET /api/datasets/{id} Детальная информация о наборе данных
POST /api/datasets Создать новый набор данных 🔒 Bearer
ПолеТипОписание
titlestringНазвание required
descriptionstringОписание required
keywordsstring[]Ключевые слова
licensestringCC-BY-4.0 | CC0-1.0 | CC-BY-NC-SA-4.0
embargoUntilISO dateДата снятия эмбарго
relatedIdsnumber[]ID связанных наборов
GET /api/datasets/{id}/export Экспорт метаданных ?format=bibtex|dc|datacite
POST /api/datasets/{id}/file Загрузка файла датасета 🔒 Bearer Binary body · X-Filename header
👤 Профиль пользователя
GET /api/me Текущий пользователь 🔒 Bearer
GET /api/my/datasets Мои наборы данных (включая черновики) 🔒 Bearer
PUT /api/my/datasets/{id} Обновить свой набор данных 🔒 Bearer
🔑 API-ключи
GET /api/keys Список активных API-ключей 🔒 Bearer
POST /api/keys Создать API-ключ (возвращается только один раз) 🔒 Bearer
{ "id": 1, "name": "jupyter-notebook", "key": "a1b2c3..." }
DELETE /api/keys/{id} Отозвать API-ключ 🔒 Bearer
🌐 OAI-PMH
GET /oai OAI-PMH endpoint — стандартный протокол метаданных verb=Identify|ListRecords|GetRecord|ListIdentifiers
ПараметрЗначениеОписание
verbIdentifyИнформация о репозитории
verbListMetadataFormatsПоддерживаемые форматы
verbListRecordsВсе записи (metadataPrefix=oai_dc)
verbGetRecordОдна запись по identifier
