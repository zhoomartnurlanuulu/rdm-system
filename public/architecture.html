Backend

Node.js v20+ · HTTP (без фреймворка)
better-sqlite3 — синхронный SQLite
SQLite FTS5 — полнотекстовый поиск
SHA-256 — хэши паролей и API-ключей

Frontend

Vanilla JS ES Modules (нет фреймворков)
CSS Custom Properties — темная/светлая тема
Fetch API — все запросы к серверу
sessionStorage — хранение JWT-токена

Протоколы

REST API — JSON, Bearer Auth
OAI-PMH — Dublin Core XML
DataCite XML — экспорт метаданных
BibTeX — экспорт для LaTeX

Безопасность

Rate Limiting — 120 req/min / 20 login
CORS — настраиваемый Origin
JWT + Refresh — 8h / 7d токены
API Keys — SHA-256 хэш, показ 1 раз
